- Ctrl + Shift + R
è una scorciatoia da tastiera che esegue principalmente un "hard refresh" (aggiornamento forzato) delle pagine web, ignorando la cache per assicurare di vedere la versione più recente del sito, ma può anche essere usata per allineare a destra il testo in Google Docs, rinominare una cartella in alcuni file manager, o attivare altre funzioni specifiche come la registrazione dello schermo in GNOME, a seconda dell'applicazione.

Ctrl+F5
e Ctrl+Shift+R (o Shift+F5) fanno la stessa cosa nei browser: eseguono un refresh forzato, ignorando la cache per scaricare una versione completamente nuova della pagina; la differenza principale è la scorciatoia, con Ctrl+Shift+R (o Shift+F5) che è spesso preferito per la sua chiara intenzione di saltare la cache, mentre F5 (o Ctrl+R) è un semplice refresh standard, che potrebbe usare la cache
